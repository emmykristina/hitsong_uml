# What Makes a Hit Song?

This project explores what characteristics define a “hit song” using unsupervised machine learning.
Instead of starting with assumptions, we let the data reveal patterns in music.

We analyze audio features from a Spotify dataset to identify groups of songs and investigate whether certain types are more likely to become popular.
